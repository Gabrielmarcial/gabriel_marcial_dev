### ***Exercicios***

1. Faça um programa que leia o nome de uma pessoa e mostre uma mensagem de boas-vindas.
1. Faça um Programa que peça um número e então mostre a mensagem O número informado foi [número].
1. Programa para somar dois números.
1. Programa para somar dois números com entrada do usuário.
1. Programa para multiplicar dois números com entrada do usuário.
1. Programa para calcular a potência de dois números com entrada do usuário.
1. Faça um programa que leia um número Inteiro e mostre na tela o seu sucessor e seu antecessor.
1. Crie um algoritmo que leia um número e mostre o seu dobro, triplo e raiz quadrada.
1. Faça um Programa que converta metros para centímetros.
1. Faça um código Python que leia a largura e altura de uma parede (em metros) e exiba a área da Parede.

1. Faça um programa que leia a largura e a altura de uma parede em metros, calcule a sua área e a quantidade de tinta necessária para pintá-la, sabendo que cada litro de tinta pinta uma área de 2 metros quadrados.

1. Faça um código Python que leia duas notas de um aluno, calcule e exiba a média aritmética das notas.
1. Faça um Programa que pergunte quanto você ganha por hora e o número de horas trabalhadas no mês. Calcule e mostre o total do seu salário no referido mês.
1. Programa Python para verificar se um número é positivo.
1. Programa Python para verificar se um número é par
