## 💼 Experiência


## Migração e modernização de plataforma de dados

Atuei em um projeto de modernização de uma plataforma de dados baseada em Databricks, realizando a migração de um ambiente legado para uma arquitetura mais moderna e governada. O trabalho envolveu a transição do Hive Metastore para o Unity Catalog, além da refatoração de pipelines e códigos existentes para melhorar desempenho, escalabilidade, segurança e governança dos dados.

---

## Plataforma de Machine Learning em Databricks

Desenvolvi pipelines completos de Machine Learning utilizando Databricks, cobrindo todas as etapas do ciclo de vida dos modelos. O projeto incluiu construção de feature stores, treinamento, versionamento, deployment, monitoramento e serving de modelos, além da implementação de práticas de MLOps para automatizar e padronizar o fluxo de desenvolvimento.

---

## Aplicações com LLMs e RAG

Participei do desenvolvimento de aplicações baseadas em Modelos de Linguagem de Grande Escala (LLMs), utilizando a arquitetura Retrieval-Augmented Generation (RAG). O projeto contemplou a criação de pipelines de embeddings, armazenamento vetorial, recuperação de documentos e integração com modelos generativos para produzir respostas contextualizadas e precisas.

---

## Arquitetura Lakehouse e Engenharia de Dados

Desenvolvi pipelines de dados em PySpark para processamento de grandes volumes de informações em ambiente Databricks. Também implementei uma arquitetura Lakehouse baseada no modelo Medalhão (Bronze, Silver e Gold), além de processos de ingestão utilizando Azure Data Factory e automações de CI/CD para garantir qualidade e confiabilidade das entregas.

---

## Modelos de previsão de custos

Desenvolvi modelos de previsão (*forecasting*) para estimar custos operacionais a partir de dados históricos. O projeto empregou técnicas de análise de dados e aprendizado de máquina para gerar previsões que apoiassem o planejamento financeiro e a tomada de decisão.

---

## Desenvolvimento de APIs e sistemas geoespaciais

Atuei no desenvolvimento de APIs REST em Python utilizando boas práticas de engenharia de software, testes automatizados e integração contínua. Também participei da modelagem e administração de bancos de dados PostgreSQL/PostGIS para armazenamento e consulta de dados espaciais utilizados em aplicações geográficas.

---

## Pipeline ETL e automação de coleta de dados

Desenvolvi processos ETL para extração automática de dados da web, transformando informações de diferentes fontes em bases estruturadas para análise. Além disso, implementei rotinas de automação utilizando Python para otimizar processos internos e reduzir atividades manuais.

---

## Dashboards para monitoramento operacional

Desenvolvi dashboards para acompanhamento de indicadores operacionais e métricas de desempenho, permitindo o monitoramento contínuo dos processos e auxiliando gestores na tomada de decisões baseadas em dados.

---

## Previsão de precipitação com visão computacional

Participei de um projeto de pesquisa voltado à previsão de chuvas utilizando imagens de radares meteorológicos e dados pluviométricos. O trabalho envolveu processamento de imagens, tratamento de dados e aplicação de algoritmos de aprendizado de máquina em Python para modelagem dos fenômenos meteorológicos.

---

## Sistema de monitoramento de contratos

Colaborei no desenvolvimento do back-end de uma plataforma web para monitoramento de contratos. O projeto foi desenvolvido em Python utilizando Flask, com testes automatizados, integração com banco de dados MySQL e foco na confiabilidade e manutenção da aplicação.

---

## Plataforma para análise de risco geográfico

Desenvolvi funcionalidades de back-end para uma plataforma destinada à análise de áreas de risco, voltada ao apoio de órgãos de defesa civil. O sistema utilizava dados geoespaciais para auxiliar no monitoramento e na avaliação de regiões suscetíveis a desastres naturais.

---

## Web Scraping em larga escala

Implementei soluções de *web scraping* para coleta automatizada de grandes volumes de dados em diferentes páginas da web. O projeto incluiu tratamento, organização e armazenamento das informações para posterior análise e integração com outros sistemas.

---

## Análise espacial de sinistros

Realizei uma prova de conceito para investigar a relação entre registros de sinistros e variáveis ambientais. O trabalho envolveu análise exploratória de dados, visualização geográfica e aplicação de técnicas de aprendizado de máquina para clusterização espacial, permitindo identificar padrões relevantes.

---

## Detecção de árvores com Deep Learning

Desenvolvi um projeto de pesquisa voltado à detecção automática de espécimes arbóreos em ambientes urbanos utilizando técnicas de Deep Learning e fusão de imagens aéreas RGB-NIR com dados LiDAR. O objetivo foi aumentar a precisão da identificação de árvores por meio da integração de diferentes fontes de dados geoespaciais, contribuindo para aplicações em sensoriamento remoto e planejamento urbano.
